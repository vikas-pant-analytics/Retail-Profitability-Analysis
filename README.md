# Retail Profitability & Discount Analysis Dashboard

## Project Overview
This project analyzes retail sales, profitability, and discount trends using Google Sheets, SQL, and Power BI.

The dashboard helps identify:
- High-profit products
- Loss-making categories
- Regional performance trends
- Discount impact on profit

---

## Tools Used
- Google Sheets
- SQL
- Power BI

---

## Key KPIs
- Total Profit
- Profit Margin
- Average Discount
- Loss Transactions

---

## Dashboard Insights
- Higher discounts are strongly associated with lower profitability.
- Copiers and Phones are the most profitable sub-categories.
- Tables and Bookcases generated major losses.
- West region achieved the highest profit.
- Consumer segment contributed the highest share of profit.

---

## Recommendations
- Reduce excessive discounting on low-performing products.
- Focus on profitable categories and regions.
- Improve pricing strategy for loss-making products.
- Use data-driven insights for business decisions.
